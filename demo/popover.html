MDPopover demo 页面

例子

代码

JS


  $(".btn").mdPopover({
    content: "this is a popover"
  });

参数设置

名称 类型 默认值 描述
onlyOne boolean false 同一时间是否只显示一个提示，设为true时，显示一个将隐藏其他的提示
content string null 设置要显示的内容，可以是文本，可以是dom标签。注意！一旦给他值则表示将有组件自动生成dom标签
related string "body" 设置popover的外部容器，popover只在容器内部出现
relatedSpace number 10 设置距外部限制容器的间距
placement static "bottom" popover出现在目标元素的位置，其他的值是"top"，"right"，"left"
