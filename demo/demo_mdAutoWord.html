MDUI AutoWord demo 页

例子

可用于自动补全，或者关键字触发提示（@谁 或者 #tag#）

代码

在需要绑定的元素上 $(selector).autoWord(config); 即可。

             $('#text').autoWord({
                key: '@',
                isTwins: false,
                listLength: 10,
                isLoopList: true,
                data: function(){
                    $.post('url','params',function(data){
                        return data;
                    });
                }});

参数设置

名称 类型 默认值 描述
key string 空字符串 约定触发符号，当用户输入的最后一个字符等于key时，则触发show block等事件。当值为空时，为全局触发。
isTwins bool false 约定当前触发符号如否成对出现，如 tag 场景。默认值为false。
listLength int 10 约定提示列表长度，默认值为10。
isLoopList bool true 约定列表是否可循环，当用户为键盘快捷键操作时，列表首和列表尾相互呼应。（即列表首上一个即为列表尾，列表尾的下一个即为列表首）
data array [] 列表数据
