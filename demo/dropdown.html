Dropdown

下拉框菜单Dropdown是一个比较常见的网页形式，明道UI也提供了dropdown供君驱使

例子

  • 已售完
  • 可购买
            
  seajs.use(["header", "dropdown", "footer"], function () {
    $("#dropdown").dropdown({
      onClick: function (li) {
        alert(li.html());
      }
    })
  });

参数

名称 类型 默认值 描述
trigger string "click" 触发Dropdown的时间名称，默认点击触发，也支持"hover"触发
onClick Function function(li) 回调函数,当点击下拉选项时触发，li,当前点击元素
onOpen Function null 回调函数，当下拉打开时触发
onClose Function null(li) 回调函数，当下拉关闭时触发
