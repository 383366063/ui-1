MDUI Dialog demo

例子

代码

            
 seajs.use("./modules/mdDialog/md.dialog", function () {
   $(".btn-dialog").mdDialog({
     title:"确定要关闭？",
     content: "this is a content ？",
     $dialog:$(".md-dialog"),
     onOK:function(){
       alert("yes")
     }
   });
 })

参数设置

名称 类型 默认值 描述
title string & html '' dialog对话的内容
content string null dialog对话的内容
width number 400 dialog的宽度
bgClose boolean true 点击背景遮罩是否关闭
$dialog JQuery Object null 如果你想自己控制dialog的DOM元素，可以配置这个，DOM元素结构见例子中的代码
onShow function null 回调函数，在box显示时调用
onHide function null 回调函数，dialog 关闭时调用
onConfirm function null 回调函数，点击dialog中的确定按钮时触发
