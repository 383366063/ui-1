MDUI Pagination demo 页

例子

ajax分页组件，根据count生成页码。当页码发生改变时，需手动在changePage事件中处理列表更新逻辑

代码

在需要绑定的元素上 $(selector).autoWord(config); 即可。

        
    $('.pageNum').Pager({
        pageIndex: 1,
        pageSize: 20, //每页显示多少条
        count: 999, //总共多少条数据
        pageCount: 5, //超过几页显示 ...
        align: 'left', //页码呈现的位置
        prev: '上一页',
        next: '下一页',
        changePage: function (pageIndex) {
            alert('当前页面为：' + pageIndex);
        }
    });

参数设置

名称 类型 默认值 描述
pageIndex int 1 约定起始页
pageSize int 20 约定页尺寸
count int 0 约定数据总长度，用于页码生成
pageCount int 5 约定 缺省参数，即连续显示几个页码
align string left 页码呈现的位置
prev string 上一页 约定上一页显示字符
next string 下一页 约定下一页显示字符
changePage function 无 约定页码改变时应该执行的事件，即重载页面数据
