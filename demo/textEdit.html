代码

DOM


<div class="doc-name edit-group">
  <div class="edit-content">
    <textarea class="input-editable" rows="2" disabled="disabled">MDRCD01和信息安全管理单和发放记录.pdf</textarea>
  </div>
  <div class="edit-action">
    <div class="group-btn group-save">
      <button class="btn btn-link btn-cancel" type="button">取消</button>
      <button class="btn btn-link btn-save" type="button">确定</button>
    </div>
    <div class="group-btn group-edit">
      <span class="btn-edit" title="修改"><i class="icon-edit"></i>修改 </span>
    </div>
  </div>
</div>

JS


$(".doc-name").mdTextEdit({
  url: "localhost:3000/modifyName"
});

参数设置

名称 类型 默认值 描述
url string null 后台验证的异步请求地址
onShow function null 回调函数，在box显示时调用
onSuccess function null 回调函数,后台验证通过时调用
onFail function null 后台验证失败时调用
onTimeout function null 没有网络时调用
