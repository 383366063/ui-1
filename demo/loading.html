MDUI Loading demo 页

例子

直接jQuery调用，这是会自动在body下添加一个loading 遮罩的div，目前因文档需要点击loading图片即可实现遮罩隐藏。

代码

不需依赖任何元素,此方法会在第一次被调用的时候在body下创建遮罩层，并通过$.toggle来实现显示隐藏，实现复用。

            seajs.use("mdLoading/md.loading", function () {
    $('.btnTriggerLoading').click(function () {
        $.fn.loading();
    });
});

参数设置

名称 类型 默认值 描述
配置参数 Object {imageSrc: '/images/folderLoading.gif', --指定加载图片路径
color: 'black', --指定遮罩层背景颜色
opacity: '0.7' --指定遮罩层透明度}
